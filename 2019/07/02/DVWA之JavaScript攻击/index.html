几个函数调用顺序及生成token的步骤如下：

1、执行token_part_1()，取phrase值并进行字符串翻转处理；

2、延迟300ms后执行token_part_2()，传入参数字符串’XX’和token值拼接并调用sha256()加密；

3、点击按钮时执行token_part_3()，将token值和字符串’ZZ’拼接并调用sha256()加密，从而得到最终的token；

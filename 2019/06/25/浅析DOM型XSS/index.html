0x01 DOM简介

DOM（Document Object Model）即文档对象模型，是HTML和XML文档的编程接口。

其中HTML DOM定义了所有HTML元素的对象和属性，以及访问它们的方法。换言之，HTML DOM是关于如何获取、修改、添加或删除HTML元素的标准。

DOM以树结构表达HTML文档。
